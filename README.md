# UGV QKD Simulation
BB84 Quantum Key Distribution protocol simulation between UGV and Laptop.

## Files
- `ugv.py` - UGV sender side (BB84 + AES-GCM video encryption)
- `laptop.py` - Laptop receiver side
- `qkd_visual_simulation.py` - Visual terminal demo

## Install Dependencies
pip install cryptography opencv-python numpy colorama

## Run
1. Start laptop.py first
2. Then run ugv.py
